Lootwhore Profile XML Layout

 

<lootwhore>

 

            <!--Settings is used to configure things not tied to specific items.  You can omit any setting, or even the entire section, and the values listed below will be used. -->

 <settings>

           

                        <!-- DefaultAction determines how Lootwhore will react to any item in treasure pool without an item-specific setting.  It can be set to lot, pass, or ignore. -->

                        <defaultaction>ignore</defaultaction>

 

                        <!-- SmartPass setting can be disabled, listonly, or everyone.  When smartpass is set to everyone, lootwhore will pass any item when anyone besides you lots it.  When set to listonly, lootwhore will pass any item lotted by anyone on your whitelist. -->

                        <smartpass>disabled</smartpass>

 

                        <!-- Rarepass setting can be disabled or enabled.  When rarepass is enabled, lootwhore will pass any rare marked item you already have on your character.  This does not count items in storage slips or NPC storage. -->

                        <rarepass>disabled</rarepass>

 

                        <!-- Zonereset setting can be disabled or enabled.  If zonereset is enabled, lootwhore will reset profile(and disable zonereset in the process) the next time you zone.  This is recommended whenever using a defaultaction of lot or pass, so you do not forget about it when going to your next content. -->

                        <zonereset>disabled</zonereset>

 

            </settings>

 

            <!--Item list is used to specify items that should have a behavior separate from the default action when in treasure pool. -->

            <itemlist>

                       

                        <!-- Items are specified by ID, but an auto generated profile will leave their name in comments for ease of editing. Each item may be set to to lot, pass, or ignore. -->

                        <item id=”1453”>lot</item> <!-- M. Silverpiece -->

                        <item id=”1452”>lot</item> <!-- O. Bronzepiece -->

                        <item id=”3497”>pass</item> <!-- Forgotten Step -->

           

            </itemlist>

 

            <!-- Unlike lootwhore in v3, drops and stores each receive their own list.  This allows you to do combinations like lot and drop or pass and store.  -->

            <!-- List items to be automatically dropped. -->

            <droplist>

                       

                        <!--Items are still specified by ID, but as there are no other attributes or content, the tag should be self closing.

                        <item id=”4249” /> <!-- S. Stratagems -->

           

            </droplist>

 

            <!-- List items to be stored.  The bags used for storing can be configured in your config xml. -->

            <storelist>

                       

                        <!--Items are still specified by ID, but as there are no other attributes or content, the tag should be self closing.

                        <item id=”1453” /> <!-- M. Silverpiece -->

                        <item id=”1452” /> <!-- O. Bronzepiece -->

                        <item id=”3497” /> <!-- Forgotten Step -->

           

            </storelist>

 

</lootwhore>
